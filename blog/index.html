Registro 1-Day CON Forense 2021

El registro para asistir como público al evento 1-Day CON Forense 2021 debe de realizarse a través del siguiente bot de Telegram. El registro es extremadamente sencillo, solo necesitas iniciar el bot con el comando /start y después utilizar el comando /registro. Pero no te preocupes que está todo documentado en el propio bot. Cuando finalizas el registro el bot te proporciona un enlace de invitación a un grupo privado. Este grupo será desde el cual se emita el evento. Leer más
